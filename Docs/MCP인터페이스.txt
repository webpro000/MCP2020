* MCP인터페이스(RestAPIService/TableLogging)
    : MLP I/F에 일 1회 약국 수집 데이터 전송
    : MLP I/F에 일 1회 병원 수집 데이터 전송
    : MLP I/F에 일 1회 맛집 수집 데이터 전송
    : 약국/병원/맛집 제공 이력 저장.
----------------------------------------------
0. 약국/병원/맛집 수집 인터페이스에서 수집된 데이터를
   PostgreSQL DB에 저장 및 Redis DB 에 저장 한다.
---------------------------------------------- 처리 요건
1. Redis DB에 저장된 약국/병원/맛집 데이터를 RESTAPI로 서비스로 각각 제공 한다.(윈도)
2. RESTAPI(json or xml) 제공 서비스는 전체/검색 목록 형태로 제공
